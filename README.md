# Monthly SKU Profit & Pricing Analysis

## Objective
Analyze SKU-level pricing and profit data to optimize pricing strategy and improve overall profitability.

## Dataset
'Sku', 'Style Id', 'Catalog', 'Category', 'Weight', 'TP', 'MRP Old', 'Final MRP Old', 'Ajio MRP', 'Amazon MRP', 'Amazon FBA MRP', 'Flipkart MRP', 'Limeroad MRP', 'Myntra MRP', 'Paytm MRP', 'Snapdeal MRP', Profit', Profit Percentage', Price Consistency, Price Band'.

## Tools Used
- Excel (Data cleaning, preprocessing, analysis)
- Python (Visualization, insights, calculated metrics)
- Power BI (Interactive dashboard)

## Process
1. Cleaned and processed raw SKU data in Excel.  
2. Added calculated columns and conducted visual analysis in Python with chart-wise insights and recommendations.  
3. Built an interactive Power BI dashboard with KPIs, slicers, and charts for decision-making.

### **KEY INSIGHTS** 
------------------
1. **Category Profitability** → Gowns lead margins, Tops lag; clear profit hierarchy across categories.  
2. **SKU Distribution** → 90%+ SKUs sit in the medium price band, showing concentration in mid‑tier pricing.  
3. **Profit vs MRP** → Strong positive correlation; higher MRPs consistently yield higher profits.  
4. **Top SKUs** → Os1109 and Os1056 variants dominate profitability, proving repeatable design/marketing success.  
5. **SKU Assortment** → Large base of 1,330 SKUs risks dilution; profitability is concentrated in fewer items.  
6. **Data Hygiene** → Missing category labels (“Nill”) and text‑based MRPs weaken analysis accuracy.

 ### **RECOMENDADTIONS**

1. Double down on gowns and replicate winning SKUs to scale profitability.
